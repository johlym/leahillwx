About This Weather Station

Collection source: WS-2902C
Tweet

Positioned on Lea Hill in Auburn, WA, this weather station is powered by an Ambient Weather WS-2902C. Data samples are taken every 5 seconds and published live to this site.

This station tracks the following weather measurements:

  • Temperature
  • Humidity
  • Barometric pressure
  • Wind
    • Direction
    • Speed
  • Rainfall
  • UV

Everything else is recorded and interpolated using software.
